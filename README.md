# Uni4Edu Website Clone

A responsive front-end clone of the Uni4Edu website built with **HTML** and **Tailwind CSS** as part of **The Gym Rwanda** gate project.

The goal of this project was to recreate the user interface of the Uni4Edu platform while demonstrating strong skills in semantic HTML, responsive design, layout techniques, and Tailwind CSS. The original platform helps students discover universities, language schools, foundation programs, and summer schools around the world. :contentReference[oaicite:0]{index=0}

---

## About The Project

This project reproduces the main pages and sections of the Uni4Edu website using only HTML and Tailwind CSS.

It focuses on:

- Pixel-perfect design recreation
- Responsive layouts for all screen sizes
- Clean and maintainable code
- Mobile-first development
- Modern UI design principles
- Reusable section structures
- Accessibility and readability
